# AlphaVault Next.js Trading App Prototype

This is the Next.js mobile app style source code with sample data included.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Included

- `app/page.tsx` - main mobile app UI with tabs/screens
- `app/globals.css` - mobile app styling
- `lib/appData.ts` - TypeScript sample data used in the UI
- `data/app-data.json` - raw sample data reference

## Features

- Login / Register UI
- Home dashboard
- Weekly / Monthly / Yearly plans
- Market watchlist
- Portfolio positions
- Wallet transactions
- Deposit / Withdraw demo actions
- Admin investor requests
- KYC and compliance screen

## Important

This is a prototype. Do not use it as a real investment platform without backend security, KYC/AML, legal terms, financial compliance, and secure payment processing.
